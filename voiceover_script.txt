"Halo semuanya, selamat pagi/siang/sore.

Tugas ini mewajibkan pembuatan sebuah website statis sederhana dan melakukan deployment ke layanan cloud gratis. Di sini, saya membangun sebuah website personal portfolio interaktif menggunakan HTML, CSS murni, dan JavaScript untuk interaksinya. 

Saat ini, kita berada di halaman utama atau Beranda. Desain website ini menggunakan tema modern dark-mode dengan warna aksen Indigo dan gradasi Rose-Red sesuai dengan panduan desain modern. Di halaman utama ini, terdapat logo personal 'wika.dev', navigasi utama, lencana 'Available for Work' dengan animasi pulsing hijau yang dinamis, serta ringkasan tentang profil saya. Terdapat pula tombol aksi 'Lihat Projek' dan 'Hubungi Saya'."


[BAGIAN 2: PROFIL & KATEGORI SKILLS - Durasi: 1:00 - 2:00]
------------------------------------------------------------------------
(Visual: Scroll ke bawah ke bagian "Profil / Tentang Saya")

"Selanjutnya, mari kita masuk ke bagian Profil atau Tentang Saya. 

Di bagian ini, saya menjelaskan secara singkat latar belakang akademik saya sebagai mahasiswa semester empat yang memiliki ketertarikan mendalam dalam arsitektur cloud computing dan full-stack development. 

Di sebelah kanan, terdapat pembagian keahlian atau skills yang dikelompokkan secara terstruktur. Pertama ada kategori 'Cloud & DevOps' yang mencakup AWS, Google Cloud Platform, GitHub Actions, Docker, Linux, dan GitHub Pages. Kedua ada kategori 'Web Development' yang meliputi HTML5, CSS3, JavaScript, Laravel PHP, Node.js, MySQL, dan Tailwind CSS. Komponen tag ini dibuat responsif dan memiliki transisi hover yang halus."


[BAGIAN 3: INFORMASI UTAMA / PROJEK UNGGULAN - Durasi: 2:00 - 3:15]
------------------------------------------------------------------------
(Visual: Scroll ke bagian "Projek Unggulan")

"Sekarang kita bergeser ke Informasi Utama, yaitu bagian Projek Unggulan.

Di bagian ini, saya menampilkan tiga buah kartu projek yang merepresentasikan eksplorasi teknis saya:
1. Pertama ada projek 'RideNusa', sebuah aplikasi penyewaan sepeda motor berbasis Laravel dan MySQL.
2. Kedua ada 'Cloud-Native Delivery', sebuah otomasi pipeline CI/CD yang men-deploy file web statis secara otomatis ke AWS S3 dan CloudFront.
3. Dan ketiga ada projek 'Serverless Visitor Analytics', sistem pemantau kunjungan menggunakan Google Cloud Functions dan Google Cloud Firestore.

Setiap kartu projek didesain interaktif dengan border glow indigo tipis ketika kursor diarahkan, memberikan feedback visual yang premium kepada pengunjung."


[BAGIAN 4: KONTAK & FITUR INTERAKTIF FORM - Durasi: 3:15 - 4:00]

"Bagian terakhir adalah Kontak. Halaman ini berisi informasi email personal, lokasi, serta tautan ke profil GitHub dan LinkedIn saya. 

Terdapat juga formulir interaktif di mana pengunjung dapat mengirim pesan. Mari kita simulasikan pengisian nama, email, dan isi pesan. Ketika saya menekan tombol 'Kirim Pesan', script JavaScript akan memvalidasi input secara lokal dan memunculkan pop-up modal sukses kustom yang sangat mulus tanpa memuat ulang (reload) halaman keseluruhan."


[BAGIAN 5: PENJELASAN DEPLOYMENT & PEMBARUAN (CLOUD) - Durasi: 4:00 - 5:00]
------------------------------------------------------------------------
(Visual: Tunjukkan repository GitHub, tab Settings Pages, dan perbandingan perubahan/update live)

"Untuk aspek Cloud Computing dari proyek ini, seluruh source code telah saya unggah ke repository GitHub publik di github.com/sastrawanwika. Deployment dilakukan secara penuh menggunakan layanan gratis GitHub Pages yang meng-host file statis langsung dari branch utama (main).

Sesuai dengan ketentuan ujian, saya juga telah melakukan pembaruan (update) terhadap website ini setelah deployment pertama berhasil. Saya menambahkan sub-kategori skill/projek baru, meng-commit perubahan, dan melakukan push kembali. Dalam hitungan detik, server GitHub Pages memicu build baru dan menampilkan pembaruan tersebut secara instan di URL publik. Hal ini mendemonstrasikan konsep dasar Continuous Deployment dalam siklus pengembangan software berbasis cloud.

Demikian presentasi dan demonstrasi proyek UAS Cloud Computing ini. Semoga dapat memenuhi seluruh kriteria penilaian dengan baik. Terima kasih atas perhatiannya, dan sampai jumpa."
