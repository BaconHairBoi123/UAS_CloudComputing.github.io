Pilihan ini didasarkan pada efisiensi teknologi awan modern yang terintegrasi langsung dengan alur kerja Continuous Deployment (CD), tanpa perlu pemeliharaan server fisik yang rumit.

Saat ini kita berada di halaman utama atau Beranda. Desain website mengusung tema Dark Mode Kosmik dengan paduan warna Neon Indigo, Violet, dan Aksen Emas. Halaman ini memuat logo 'wika.dev', navigasi melayang (floating pill), ringkasan profil, serta fitur interaktif multi-klik pada logo: 
1. Klik 1 kali memutar logo dan meluncurkan roket luar angkasa 🚀.
2. Klik 2 kali memicu ledakan Supernova dan efek Kecepatan Cahaya Warp Speed ✨.
3. Klik 3 kali memicu fenomena Gargantua Black Hole 🕳️ yang menyedot elemen-elemen web secara 3D sebelum kembali normal."

Untuk menyajikan daftar projek secara rapi, saya mengimplementasikan sistem paginasi dinamis berbasis JavaScript. Di halaman pertama, terdapat 4 kartu projek unggulan:

[BAGIAN 4: KONTAK & AMBIENT SPACE AUDIO - Durasi: 3:30 - 4:15]
------------------------------------------------------------------------
(Visual: Scroll ke bagian "Contact", tunjukkan form interaktif dan klik tombol pemutar audio ambient di sudut kiri bawah)

"Bagian berikutnya adalah Kontak. Halaman ini memuat tautan email personal serta media sosial GitHub dan LinkedIn. 

Formulir kontak ini dilengkapi dengan validasi JavaScript lokal dan modal pop-up sukses kustom tanpa memuat ulang (reload) halaman. Selain itu, di sudut kiri bawah layar terdapat tombol pemutar audio ambient luar angkasa (`#audio-toggle`). Apabila audio mp3 tidak dimuat, sistem memiliki fallback otomatis berbasis Web Audio API Synthesizer kustom untuk menghasilkan efek suara kosmik."


[BAGIAN 5: PENJELASAN DEPLOYMENT & CONTINUOUS DEPLOYMENT (CD) - Durasi: 4:15 - 5:00]
------------------------------------------------------------------------
(Visual: Tunjukkan repository GitHub 'UAS_CloudComputing.github.io', tab Settings Pages, dan live website)

"Seluruh kode sumber proyek ini disimpan dalam repositori GitHub dan di-deploy secara terotomatisasi ke URL publik `baconhairboi123.github.io/UAS_CloudComputing.github.io/` menggunakan GitHub Pages.

Sebagai pembuktian Continuous Deployment (CD), setiap kali pembaruan seperti penambahan sistem paginasi dinamis di-push ke branch utama (`main`), infrastruktur awan GitHub Pages secara otomatis memicu proses build ulang dan memperbarui tampilan live site dalam hitungan detik.

Penggunaan arsitektur Serverless Cloud ini terbukti sangat efisien, cepat, aman, dan memangkas biaya operasional server. Sekian demonstrasi proyek UAS Cloud Computing ini. Terima kasih atas perhatiannya."
