Tugas ini mewajibkan pembuatan sebuah website statis sederhana dan melakukan deployment ke layanan cloud gratis. Di sini, saya membangun sebuah website personal portfolio interaktif menggunakan HTML, CSS murni, dan JavaScript untuk interaksinya. 

Saat ini, kita berada di halaman utama atau Beranda. Desain website ini menggunakan tema modern dark-mode dengan skema warna kosmik—menggunakan gradasi Biru Neon, Indigo, dan Ungu Gelap sesuai dengan tren desain premium saat ini. Logo 'wika.dev' ini juga interaktif: jika diklik sekali, ia akan memutar logo dan meluncurkan roket 🚀; sedangkan jika diklik dua kali, ia memicu kilatan supernova yang mendemonstrasikan efek warp kecepatan cahaya pada latar belakang bintang."

[BAGIAN 3: PROJEK UNGGULAN & PAGINASI - Durasi: 2:00 - 3:30]
------------------------------------------------------------------------
(Visual: Scroll ke bagian "Highlight Projects", tunjukkan 4 projek di Halaman 1, lalu klik halaman 2 untuk menunjukkan Projek 5)

"Sekarang kita berpindah ke bagian Projek Unggulan.

Karena terdapat lebih dari 4 projek, saya telah mengimplementasikan sistem paginasi dinamis berbasis JavaScript. Di halaman pertama, terdapat 4 kartu projek unggulan:
1. Pertama ada 'RideNusa Web', platform rental motor berbasis Laravel dan MySQL.
2. Kedua ada 'Hello World', web bertema penghijauan edukatif menggunakan HTML, CSS, dan JavaScript.
3. Ketiga ada 'RideNusa Mobile', aplikasi mobile berbasis Flutter, Dart, dan REST API.
4. Keempat ada 'KRS Universitas Touhou Indonesia', portal kartu rencana studi interaktif dengan Next.js 14, raw PHP backend, dan MySQL.

Setiap kartu didesain dengan efek hover interaktif dan scrolling gambar pratinjau yang halus. Jika kita mengklik tombol halaman '2' pada navigasi paginasi di bagian bawah, sistem akan beralih secara mulus dan menampilkan projek kelima, yaitu placeholder 'Coming Soon' untuk projek masa depan."


[BAGIAN 4: KONTAK & FITUR INTERAKTIF FORM - Durasi: 3:30 - 4:15]
------------------------------------------------------------------------
(Visual: Scroll ke bagian "Contact", simulasikan pengisian form secara cepat, lalu klik submit)

"Bagian berikutnya adalah Kontak. Halaman ini memuat email personal serta tautan media sosial seperti GitHub dan LinkedIn. 

Terdapat juga formulir interaktif di mana pengunjung dapat mengirim pesan. Mari kita simulasikan pengisian formulir ini. Ketika tombol 'Kirim Pesan' ditekan, JavaScript akan memvalidasi input secara lokal dan memunculkan pop-up modal sukses kustom yang dinamis tanpa memuat ulang (reload) halaman, menjaga pengalaman pengguna tetap mulus."


[BAGIAN 5: PENJELASAN DEPLOYMENT & CD (CLOUD) - Durasi: 4:15 - 5:00]
------------------------------------------------------------------------
(Visual: Tunjukkan repository GitHub, tab Settings Pages, dan website live)

"Seluruh kode sumber proyek ini disimpan dalam repositori GitHub dan di-deploy secara otomatis ke URL publik menggunakan layanan cloud gratis GitHub Pages. 

Sebagai pemenuhan kriteria Continuous Deployment, pembaruan konten seperti penambahan paginasi projek dan kartu projek baru langsung di-push ke branch utama. GitHub Pages secara otomatis memicu proses build ulang dan memperbarui website publik dalam hitungan detik.

Sekian demonstrasi proyek UAS Cloud Computing ini. Semoga dapat memenuhi seluruh kriteria penilaian dengan baik. Terima kasih atas perhatiannya."
